SQL Activities

In this assignment, I learned how to execute SQL statements and incorporate them into HTML files. I practiced creating valid HTML pages, including external styling, and utilized SQL examples from w3schools.org to understand various SQL commands like SELECT, WHERE, COUNT, INSERT INTO, and UPDATE. These activities helped me grasp the fundamentals of SQL usage within web development and how to integrate SQL statements into HTML documents effectively.

SQL Activity Examples

  • SQL SELECT Statement

    SELECT * FROM CUSTOMERS;

  • SQL WHERE Clause

    SELECT * FROM Customers WHERE Country = "USA";

  • SQL COUNT Function

    SELECT COUNT(CustomerID) AS "Total Customers" FROM Customers;

  • SQL INSERT INTO Statement

    INSERT INTO Customers (CustomerName, ContactName, Address, City, PostalCode, Country) VALUES ('Cardinal', 'Tom B. Erichsen', 'Skagen 21', 'Stavanger', '4006', 'Norway');

  • SQL UPDATE Statement

    UPDATE Customers SET ContactName='Alfred Schmidt', City='Hamburg' WHERE CustomerID = 1;
